Welcome to the Green Button Alliance Data Custodian

The Green Button Alliance has, with the support of the open source community, prepared a reference Green Button Data Custodian implementation. This implementation supports a full complement of Green Button facilities and, as an Open Source project, is freely available for download by any interested parties.

Data Custodian Services

Green Button Data Custodians may, with the authorization of the Retail Customer, publish energy and resource usage information to registered Green Button Third Party Service Providers. These published information streams allow Third Parties to provide new and innovative services to energy consumers!

Developers Guide

The Green Button Developers Guide provides both overview and detailed information on the mechanisms of Green Button. You can learn about Atom feeds and the OData standard from which Green Button was derived, as well as the details of the Energy Usage Information contained in Green Button feeds.

API Reference

Green Button services are also available via RESTful APIs. APIs allow the developer to use the Green Button Alliance Data Custodian and Third Party servers to develop new applications, providing services and features to better engage the Retail Customer to understand their energy usage.
